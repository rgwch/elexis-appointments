# Elexis-Appointments frontend

The frontend is a Svelte/Vite/Typescript App with internationalization via swvelte-i18n.
The translations are all AI-Generated (Github Copilot using Claude Sonnet 4.5).

## Prerequisites

bun >= 1.3.6 or node >= 22.15

## Launch development mode

`bun run dev` or `npm run dev`

## Build for production

`bun run build` or `npm run build` 

## Customization (as needed)

* Edit index.html
* Change the site icon in public/app.png
* Modify texts in src/lib/i18n/*.json
* Modify texts in src/lib/content/*.md
